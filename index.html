Как это работает

  • Регистрация Вы вводите в систему данные вашей компании
  • Выбор решения
    1. создаете онлайн-витрину или магазин
    2. используете виджет оплаты на любом вашем интернет-ресурсе
    3. берете готовый модуль интеграции с CMS вашего сайта
    4. встраиваете сервис на ваш сайт через API
  • Синхронизация с 1С С нашей помощью (или без нее) настраиваете обмен данными из системы учета торговли 1С: Предприятие
  • Оплата на сайте Ваш клиент производит оплату через платежные модули Pokupo. Все платежи проходят через онлайн-кассу Pokupo
  • Формирование и отправка чека При успешной оплате формируется онлайн чек, он отправляется на электронную почту вашего клиента, а информация о платеже поступает в налоговую
